## 一、诗人概况

&emsp;&emsp;夏宇，本名黄庆绮，笔名李格弟、童大龙。1956年生，毕业于国立艺专影剧科。十九岁开始写诗，是一位发迹甚早的女诗人，早期曾参加草根诗社，诗集未出之时，即入选1981年由张默所编的《现代女诗人选集》可见其受瞩目的程度，且在1984年获得《中外文学》举办第一届现代诗选拔第二名。
