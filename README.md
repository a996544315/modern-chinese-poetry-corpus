## 符号
#### 分隔符

> =深圳\*的+嫩叶\*突然-成熟&。

+ \+ 名词性意象词汇
+ \- 意象形容词修饰定语
+ $ 量词
+ { 意象被动词修饰定语
+ } 意象组合谓语（被动主谓）
+ [n 意象组合谓语（主谓），数字n为意象词汇坐标，如n=1表示该词语与前1个出现被+标记的【名词性意象词汇】进行组合，n为负数则向后。n可不写，默认为1。
+ ]n 意象组合谓语（谓宾），n的取值与上述相反，正数往后，负数向前，默认为1.
+ = 无隐喻或约定成俗的未登录名词（地名、人名、组织名等）
+ \# 其他形容词、副词
+ \* 其他介词、连词、人称代词
+ & 标点符或者数字
+ ^ 章节号
